The Most Important Anchor Attributes

There are three anchor attributes you need to know to create functional hyperlinks. These attributes are `href`, `target`, and `download`.

Specify a Hyperlink Target: `href` The hypertext reference, or href, attribute is used to specify a target or destination for the anchor element. It is most commonly used to define a URL where the anchor element should link to. > In this example, the `anchored text` > links to the URL `www.example.com`. An `href` can do a lot more than just link to another website.
  • can be used to link directly to any element on a web page that has been assigned an id.
  • can be used to link to a resource using a protocol other than http.
  • It can be used to run a script.
